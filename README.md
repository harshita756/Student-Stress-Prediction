# Student Stress Prediction System

## Introduction
The Student Stress Prediction System is a Machine Learning project that predicts the stress level of students based on responses to a mental health survey. The project provides an interactive Gradio interface for easy stress prediction.

## Problem Statement
Students often experience stress due to academic, personal, and social factors. Early prediction of stress levels can help identify students who may need support.

## Proposed Solution
This project uses a Machine Learning model trained on a mental health survey dataset to predict student stress levels based on user responses.

## Dataset Used
- MentalHealthSurvey.csv

## Technologies Used
- Python
- Google Colab
- Gradio

## Features
- Interactive Gradio interface
- Stress level prediction
- Easy-to-use questionnaire

## Results
The model predicts the stress level based on the user's responses through the Gradio interface.
